This project uses **Yarn** and requires **Node.js v20+**.

Install dependencies:

```bash
yarn install
```

Run development server:

```bash
yarn dev
```

Build production version:

```bash

```

Run linters:

```bash
yarn lint
yarn lint:styles
yarn lint:styles:fix
```
